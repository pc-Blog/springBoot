# 个人博客网站 — 功能需求文档

## 项目定位
面向程序员的个人技术博客，核心目标：**展示学习成长轨迹** + **项目作品集**。

## 已确认的关键决策
| 决策项 | 选择 |
|--------|------|
| 评论系统 | 需要，用户需登录后评论 |
| SEO | 不做 SEO，纯 SPA 客户端渲染 |
| 部署环境 | 单一云服务器（前端 + 后端 + 数据库同一台） |
| 内容初始化 | 从零开始，无需迁移 |
| 外部数据源 | 本地 Tomcat 服务器提供文学作品 + 音乐资源 |
| 数据集成方式 | SpringBoot 后端代理 Tomcat API，统一接口输出给前端 |
| 前端框架 | Next.js + TypeScript |
| 后端框架 | SpringBoot + Java 21 |
| 用户体系 | 访客可注册登录，管理端接口需JWT鉴权 |

---

## 一、技术栈

### 1.1 前端 (Next.js 14+)
- TypeScript — 类型安全
- App Router — 文件系统路由
- Axios — HTTP 请求、拦截、统一错误处理
- Tailwind CSS — 原子化 CSS
- Zustand — 轻量状态管理
- react-markdown + rehype-highlight — Markdown 渲染 + 代码高亮
- CSS Variables + Context — 暗色/亮色主题切换

### 1.2 后端 (SpringBoot 3.x + Java 21)
- MyBatis-Plus — ORM, 表前缀 t_, 逻辑删除
- PostgreSQL 16 — 主数据库
- MinIO — 图片/文件对象存储
- Spring Security + JWT — 认证鉴权
- WebClient — 代理 Tomcat API
- BCrypt — 密码哈希
- FastJSON — JSON 序列化
- Lombok — 代码简化

### 1.3 外部服务 (Docker)
- bg-postgres:15432 — PostgreSQL
- bg-redis:16379 — Redis
- bg-minio:19090 — MinIO (bucket: blog, public read)
- bg-tomcat:18016 — Tomcat (文学作品+音乐)
- bg-nginx:10050 — Nginx

## 二、数据库设计 (11表)

```
t_user         — 用户(注册/登录/评论)
t_category     — 分类(ARTICLE / PROJECT)
t_tag          — 文章标签
t_article      — 博客文章
t_article_tag  — 文章↔标签 多对多关联
t_project      — 项目作品
t_timeline     — 学习时间线
t_skill        — 技能图谱
t_about        — 关于我(单行)
t_comment      — 文章评论
t_media        — 上传文件(MinIO)
```

所有表使用 `deleted` 字段逻辑删除(0=正常, 1=删除), MyBatis-Plus 全局自动过滤。

## 三、访客端功能需求

### 3.1 文章模块（核心）
| 功能 | 说明 |
|------|------|
| 文章列表 | 分页展示、仅已发布文章、置顶优先→时间倒序 |
| 按分类筛选 | 选择分类后仅显示该分类文章 |
| 按标签筛选 | 选择标签后仅显示含该标签文章 |
| 关键词搜索 | 搜索 title + summary，后端 LIKE 匹配 |
| 文章详情 | Markdown 渲染、代码高亮、TOC 目录导航 |
| 前后导航 | 详情页显示上一篇/下一篇文章链接 |
| 封面图 | 文章列表卡片展示封面缩略图 |
| 阅读量 | 每次访问详情自动 +1 |
| 分类名/标签名 | 列表和详情均显示中文名(非ID) |
| 评论数 | 列表显示评论总数 |

### 3.2 项目展示模块
| 功能 | 说明 |
|------|------|
| 项目列表 | 卡片式布局，缩略图、项目名、简述、技术栈标签 |
| 按分类筛选 | 选择 PROJECT 分类后过滤 |
| 项目详情 | Markdown 内容、技术架构、GitHub 链接、在线 Demo 链接 |
| 前后导航 | 详情页显示上一个/下一个项目 |
| 封面图 | 通过 Media 模块上传后引用 |

### 3.3 学习轨迹模块
| 功能 | 说明 |
|------|------|
| 学习时间线 | 时间轴展示里程碑，按时间倒序 |
| 技能图谱 | 技术栈熟练度展示(进度条/图表)，按熟练度倒序 |

### 3.4 关于我
| 功能 | 说明 |
|------|------|
| 个人简介 | Markdown 格式自我介绍 |
| 联系方式 | 邮箱、GitHub 主页 |
| 社交链接 | JSON 数组 [{name,url}]，图标链接展示 |

### 3.5 文学作品展示 (数据来源：Tomcat)
| 功能 | 说明 |
|------|------|
| 作品列表 | 分页展示诗词/散文，按体裁分类 |
| 作品详情 | 沉浸式阅读体验，优雅排版 |
| 体裁筛选 | 诗词、散文、其他文学形式切换 |
| 搜索 | 按标题/作者/内容关键词搜索 |

数据流：前端 → SpringBoot(代理) → Tomcat → 返回 JSON  
Tomcat 不可用时前端显示降级提示，不影响博客核心功能。

### 3.6 音乐播放 (数据来源：Tomcat)
| 功能 | 说明 |
|------|------|
| 迷你播放器 | 页面底部/侧边固定，仅展示曲名、歌手 |
| 随机播放 | 随机切歌，无上一首/列表/循环控制 |
| 播放/暂停 | 单按钮切换 |
| 封面小图 | 播放器旁展示当前曲目封面 |

数据流：前端 → SpringBoot(代理) → Tomcat(返回 MinIO URL)  
音频播放：前端 `<audio>` 直接加载 MinIO URL

### 3.7 数据看板
| 功能 | 说明 |
|------|------|
| 综合统计 | 已发布文章数、项目数、技能数、时间线节点数、评论总数 |
| 总阅读量 | 所有文章阅读量之和 |
| 热门文章 | 阅读量 Top5 文章 |
| 最新评论 | 最新 5 条评论 |

### 3.8 用户系统
| 功能 | 说明 |
|------|------|
| 注册 | 用户名+密码+可选昵称邮箱注册 |
| 密码登录 | 用户名+密码，返回 JWT(7天有效) |
| GitHub 登录 | Github OAuth，首次自动创建账号 |
| 评论 | 登录后才能发表评论，自动关联用户身份 |

### 3.9 交互功能
| 功能 | 说明 |
|------|------|
| 评论 | 登录用户可评论，autherName/Email 自动从用户信息填充 |
| 响应式布局 | PC + 移动端适配 |
| 主题切换 | 暗色/亮色模式 |

## 四、管理端功能需求

> 管理端接口均需 JWT 鉴权。文学作品和音乐在 Tomcat 侧管理。

### 4.1 内容管理
| 功能 | 说明 |
|------|------|
| 文章管理 | 新建/编辑/删除，Markdown 编辑器 |
| 文章草稿 | isPublished=0，管理端可见，访客端不可见 |
| 文章发布 | 草稿→发布，自动设置 createdAt |
| 文章下架 | 发布→草稿 |
| 文章置顶 | 切换 isPinned，置顶文章在列表最前 |
| 项目管理 | 新建/编辑/删除项目，发布/下架 |
| 时间线管理 | 添加/编辑学习里程碑 |
| 技能管理 | 添加/编辑技能(名称+熟练度) |
| 分类管理 | 维护文章分类(ARTICLE)和项目分类(PROJECT) |
| 标签管理 | 维护文章标签 |
| 关于我编辑 | 编辑个人简介、联系方式、社交链接 |

### 4.2 媒体管理
| 功能 | 说明 |
|------|------|
| 图片上传 | 上传到 MinIO，返回 URL，限制 10MB |
| 类型限制 | 允许 jpeg/png/gif/webp/svg/pdf/zip |
| 文件浏览 | 分页查看已上传文件 |
| 文件删除 | 删除前检查引用(被文章/项目引用则禁止删除) |
| 批量删除 | 逐文件检查，汇总成功/失败 |
| 图片嵌入 | Markdown 中用 ![alt](MinIO_URL) 嵌入，MinIO 已设 public read |

### 4.3 校验规则
| 规则 | 说明 |
|------|------|
| 注解校验 | @NotBlank/@Size/@Email/@NotNull/@Min/@Max，中文错误提示 |
| 用户名唯一 | 创建/更新时查重，排除软删除+排除自身 |
| 标签名唯一 | 同上 |
| 技能名唯一 | 同上 |
| 分类名同type唯一 | 同 type 内不可重名，不同 type 允许同名 |
| 文章分类校验 | categoryId 对应的分类必须存在且 type=ARTICLE |
| 项目分类校验 | categoryId 对应的分类必须存在且 type=PROJECT |
| 评论文章校验 | articleId 对应的文章必须存在且已发布 |

## 五、Tomcat 集成

| 决策项 | 选择 |
|--------|------|
| 部署关系 | Tomcat 与 SpringBoot 同一台机器 |
| 博客代理 | SpringBoot WebClient 代理三接口 |
| 文学接口 | 体裁列表 / 作品分页(支持筛选关键词) |
| 音乐接口 | 随机返回一首曲目(曲名/歌手/封面/MinIO URL) |
| 音乐存储 | Tomcat 存 MinIO，博客代理返回 URL |
| 音频播放 | 前端 `<audio>` 直连 MinIO，不经过后端 |

```
Next.js → SpringBoot(OpController) → Tomcat(文学+音乐)
Next.js → MinIO(音频直连)
```

## 六、非功能性需求

- **性能**：图片懒加载、音频流式加载
- **安全**：防 XSS(react-markdown)、JWT 鉴权、密码 BCrypt 哈希
- **可维护**：前后端分离，三层代码规范(Mapper/Service/Controller)
- **容错**：Tomcat 不可用时文学/音乐模块优雅降级
- **日志**：所有 Controller 方法 @Slf4j 记录请求参数
- **统一错误**：GlobalExceptionHandler 返回中文错误信息
